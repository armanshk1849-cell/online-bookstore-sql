#  Online Bookstore SQL Analysis

##  Project Overview
This project analyzes an online bookstore database using SQL to extract meaningful business insights.

## Tools Used
- PostgreSQL
- SQL

##  Key Features
- Used JOINs to combine multiple tables (Books, Orders, Customers)
- Applied GROUP BY and HAVING for aggregations
- Used COALESCE to handle NULL values
- Implemented DISTINCT queries
- Performed data analysis on sales, customers, and inventory

##  Key Insights
- Identified customers with multiple orders
- Found most frequently ordered books
- Calculated total sales by city and author
- Analyzed stock remaining after fulfilling orders
- Determined highest spending customers

##  Database Structure
- Books
- Customers
- Orders

- Data aggregation using GROUP BY
- Handling NULL values with COALESCE
